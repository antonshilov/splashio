# Splashio
Unsplash client for Android
[![Build Status](https://travis-ci.com/antonshilov/splashio.svg?token=hbYjrhMxApzChtSYo7nL&branch=master)](https://travis-ci.com/antonshilov/splashio)

## Build instructions
* clone repo
* run `./gradlew assembleDebug`
* You're good to go! :)
